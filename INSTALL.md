# Shield Installation Guide

## Install Dependencies

### Backend Dependencies
```bash
cd backend
npm install
```

### Frontend Dependencies
```bash
npm install
```

## Quick Start

1. **Set up backend environment:**
```bash
cd backend
cp .env.example .env
# Edit .env with your settings
```

2. **Start backend:**
```bash
cd backend
npm start
```

3. **Start frontend (in another terminal):**
```bash
npm run dev
```

## All-in-One Install Command

From project root:
```bash
cd backend && npm install && cd .. && npm install
```
